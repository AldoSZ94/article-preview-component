# 📰 Article Preview Component

Proyecto de maquetación web basado en el desafío **"Article Preview Component"** de Frontend Mentor.  
El objetivo fue construir un componente interactivo de vista previa de artículo con un sistema de compartir contenido (*share overlay*) utilizando HTML, SCSS y JavaScript moderno.

---

## 📌 Descripción

Este proyecto consiste en un componente de artículo con diseño responsive que incluye información del autor y un botón para compartir contenido en redes sociales.

Al interactuar con el botón **Share**, se despliega un overlay con iconos sociales, proporcionando una experiencia interactiva y adaptada a diferentes tamaños de pantalla.

Se puso especial énfasis en:

- Estructura semántica  
- Diseño responsive  
- Interactividad con JavaScript  
- Accesibilidad web  
- Código limpio y escalable  
- Organización modular de estilos con SCSS

## 🎯 Características principales

- 📱 Diseño completamente responsive para mobile y desktop  
- 🔄 Overlay interactivo para compartir contenido  
- 🖱️ Interacción dinámica mediante eventos de JavaScript  
- 🎨 Transiciones suaves para mejorar la experiencia visual  
- 🏷️ Uso de metodología **BEM** para una estructura CSS escalable  
- ♿ Implementación de accesibilidad con `aria-label` y `aria-expanded`  
- 🧩 Organización modular de estilos con **SCSS**  
- ⚡ Entorno de desarrollo moderno con **Vite**

## 🧠 Aprendizaje

Durante este proyecto puse en práctica conceptos clave como:

- Manipulación del DOM con JavaScript  
- Uso de `addEventListener()` para manejar eventos  
- Implementación de `classList.toggle()` para interactividad dinámica  
- Manipulación de atributos con `setAttribute()`  
- Uso de `aria-expanded` para mejorar accesibilidad  
- Diseño responsive con media queries  
- Creación de overlays interactivos con SCSS  
- Organización de estilos mediante metodología BEM  
- Uso de transiciones CSS para mejorar la experiencia de usuario  
- Desarrollo de interfaces modernas con Vite
